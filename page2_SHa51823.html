Maze Maker


This is a project that I had made just for myself. The Maze Maker block starts from the top left corner and starts building a maze. It randomly travels to any available neighbour of the current cell. If it encounters a position, where there is no neighbour available, it traces it's way back. I have used arrays for this movement. When every cell is traversed, or in other words, the maze is complete, the block returns back
